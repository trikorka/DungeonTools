# [Dungeon Tools](https://trikorka.github.io/DungeonTools/) 🎲

**Dungeon Tools** — это набор интерактивных веб-инструментов для мастеров подземелий (ГМов) и игроков в настольные ролевые игры. Проект предоставляет удобные цифровые решения для управления игровыми сессиями: от создания токенов до отслеживания инициативы в бою.

![Dungeon Tools Preview](assets/images/logo.svg)

---

## ✨ Особенности проекта

- 🧩 **Модульная структура** — все инструменты собраны на одной странице, каждый доступен в один клик.
- 🎨 **Современный UI** — стильный дизайн с использованием Bootstrap 5, тёмная тема, адаптивная верстка.
- ⚡ **Быстрый старт** — интуитивно понятный интерфейс, подсказки для новых пользователей.
- 🔮 **Поддержка популярных НРИ** — D&D, Call of Cthulhu, Daggerheart и другие системы.
- 🆓 **Бесплатный базовый доступ** — расширенные функции доступны по подписке.

---

## 🛠️ Доступные инструменты

| Инструмент | Описание |
|-----------|----------|
| **Токенатор** | Создание и управление токенами персонажей и монстров |
| **Лист персонажа** | Редактируемые цифровые листы персонажей |
| **Цифровые кубы** | Генератор случайных бросков (d4, d6, d20 и др.) |
| **Трекер инициативы** | Управление порядком хода в бою |
| **Виртуальный стол** | Совместная работа над картами и персонажами в реальном времени |
| **В разработке** | Скоро появятся новые функции |
